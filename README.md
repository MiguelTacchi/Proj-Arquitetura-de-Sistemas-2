Descrição

Este projeto implementa um sistema de pagamentos em Java que simula um caixa de loja, permitindo ao usuário escolher entre diferentes formas de pagamento:

 Cartão de Crédito
 Cartão de Débito
 PIX
 Dinheiro

O objetivo principal é aplicar os princípios GRASP (General Responsibility Assignment Software Patterns) para organizar melhor as responsabilidades do sistema.
